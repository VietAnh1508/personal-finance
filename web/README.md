# Web App Setup (US-WEB-000)

This folder contains the React SPA baseline for web migration.

## Prerequisites

- Node.js 20+
- pnpm 10+

## Local development

```bash
cd web
pnpm install
pnpm dev
```

## Standard scripts

- `pnpm dev` start Vite dev server
- `pnpm build` run typecheck build and create production bundle
- `pnpm lint` run ESLint
- `pnpm typecheck` run TypeScript checks
- `pnpm test` run Vitest once
- `pnpm test:watch` run Vitest in watch mode
