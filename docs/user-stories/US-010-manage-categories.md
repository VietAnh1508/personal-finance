# US-010: Manage Categories

## User Story
As a user,
I want to manage categories,
so that transactions can be classified meaningfully.

## Acceptance Criteria
1. User can view predefined categories (initial examples include Groceries, House renting, Transportation).
2. User can add and edit categories.
3. User can archive categories (not hard delete).
4. Archived categories remain linked to historical transactions.
5. Archived categories are hidden by default and can be shown via toggle.
