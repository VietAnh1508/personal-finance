# US-009: Edit and Delete Transactions

## User Story
As a user,
I want to edit or delete transactions,
so that I can correct mistakes.

## Acceptance Criteria
1. User can open transaction detail from transaction list.
2. User can edit transaction fields and save changes.
3. User can delete transaction with confirmation.
4. For transfer transactions, editing one side updates both linked rows.
5. For transfer transactions, deleting one side deletes both linked rows.
