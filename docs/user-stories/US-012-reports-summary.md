# US-012: View Report Summaries

## User Story
As a user,
I want to view report summaries for selected periods,
so that I can understand my financial performance.

## Acceptance Criteria
1. Reports support month, year, and custom date range.
2. Reports support wallet filter: `All Wallets` or individual wallet.
3. Report returns total income, total expense, and net income.
4. Transfer and adjustment transactions are excluded from income/expense totals.
5. Future-dated transactions are included if within selected report range.
